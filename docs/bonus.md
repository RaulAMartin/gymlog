# Bonus implementados

## Animaciones

Se han añadido animaciones a la interfaz usando Motion para React.

La librería se instaló en el frontend con:

```bash
npm install motion
Componentes animados

Se han animado:

páginas principales mediante AnimatedPage
tarjetas de ejercicios
tarjetas de sesiones
tarjetas de recomendaciones RM
Archivo AnimatedPage

El componente principal de animación está en:

client/src/components/AnimatedPage.tsx

Este componente aplica una animación de entrada suave a las páginas.

Objetivo

El objetivo de las animaciones es mejorar la experiencia de usuario sin afectar a la funcionalidad principal de la aplicación.

Bonus aplicado

Este apartado cumple el bonus de añadir animaciones al proyecto.
